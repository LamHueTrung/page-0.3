Một Số lưu ý khi viết code các bạn cần biết: 1.Ngắn gọn, dễ nhìn, dễ hiểu. 
                        
int main() {
    printf("welcome Tự học lập trình cùng LHT");
    return 0;
}
// nhìn vào đoạn code bạn sẽ nhìn ra code để in ra màn hình 
dòng chữ "welcome Tự học lập trình cùng LHT"
Lưu ý: ngắn gọn nhưng phải đầy đủ ý, tránh ngắn quá mức đến nổi không hiểu được code mô tả điều gì. 2.đơn giản, cụ thể. 
                        
// có nhiều cách viết code nhưng hãy chọn cách nào đơn giản và hiệu quả nhất.
Lưu ý: hãy bắt đầu dùng với những câu lệnh đơn giản, tránh dùng các câu lệnh nâng cao khi chưa nắm rõ nguyên lí và công dụng. 3.Tránh lặp lại đoạn code quá nhiều . 
                        
int main() {
    int a;
    printf("nhap a");
    scanf("%d", &a);
    printf("gia tri của a %a", a);
    int b;
    printf("nhap b");
    scanf("%d", &b);
    printf("gia tri của b %b", b);
    int c;
    //đoạn lệnh này 
    //có cấu trúc giống nhau 
    //và bị lặp lại 3 lần
    printf("nhap c");
    scanf("%d", &c);
    printf("gia tri của c %c", c);
    /////////////////
    return 0;
}
Lưu ý: đối với những đoạn code giống nhau các bạn nên viết dưới dạng funcion để tránh việc lặp lại 1 đoạn code quá nhiều dòng như ví dụ trên (cách dùng funcion mình sẽ nói trong những video sau). 4.Làm tới đâu chắc tới đó . 
                        
// làm tới đâu các bạn kiểm tra tới đó,
 chắc rồi mới viết tiếp.
Lưu ý: khi viết xoq 1 đoạn code thực hiện 1 công việc các bạn nên chạy thử xem code đã hoạt động ổn định chưa rồi mới tiếp tục viết code để thực hiện đoạn chương trình tiếp theo.
